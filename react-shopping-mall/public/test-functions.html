📊 구현된 기능 상세

  • 회원가입: 폼 유효성 검사, 실시간 비밀번호 강도 체크
  • 로그인: Mock 인증 시스템, 관리자 권한 구분
  • 상품 관리: 카테고리별 분류, 필터링, 정렬
  • 장바구니: 로컬 스토리지 저장, 수량 조절
  • 결제: 다단계 결제 프로세스, 배송 정보 입력
  • 관리자: 상품 추가, 대시보드, 권한 기반 접근 제어

🔧 기술 스택

  • 프론트엔드: React 19.1.0, React Router 7.6.1
  • 상태 관리: Context API (AuthContext, CartContext)
  • 스타일링: CSS Modules, React Icons
  • 개발 도구: Create React App, ESLint, Prettier
